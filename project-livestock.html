.NET MAUI, C#, XAML, MVVM, SQLite

Project Overview

A cross-platform livestock management app for managing cow and sheep records, calculations, filtering, sorting, and dashboard analysis.

Key Features / Responsibilities

  • Built a cross-platform livestock management app using MVVM, local SQLite storage, and repository/service layers for CRUD, search, filtering, and sorting of cow and sheep records.
  • Implemented calculation logic for income, tax, cost, profit, average weight, and profit forecasting.
  • Developed dashboard and analysis features with chart visualisations using Microcharts and SkiaSharp.
  • Structured the app with models, views, view models, services, and repositories to improve separation of concerns and maintainability.
  • Used local seed data and SQLite database handling to simulate a practical data-driven mobile/desktop application.

Technical Implementation

The app separates UI, view models, services, repositories, and SQLite storage. Data operations run through repository/service layers, while dashboard calculations and chart visualisations summarise livestock records for practical analysis.

What I Learned

Cross-platform app structure, MVVM, local SQLite storage, repository/service patterns, data calculations, and dashboard visualisation.
